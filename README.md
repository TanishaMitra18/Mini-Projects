# Mini-Projects

This repository contains a series of small projects and analysis across multiple technologies:
- Excel automation
- Power BI dashboards
- SQL queries and data manipulation
- VBA macros
- Python scripts
- AI-based solutions

## Structure
Each project is maintained in its own branch:
- 'Personal-Task-Tracker' → Automates task tracking in Excel with VBA and progress visualization | [View Project](https://github.com/TanishaMitra18/Mini-Projects/tree/Personal-Task-Tracker/Personal-Task-Tracker)
- 'Heart Disease Exploratory Data Analysis' → Exploratory data analysis on the Cleveland Heart Disease dataset, identifying key cardiovascular risk factors through normalized analysis, hypothesis testing, and correlation mapping using Python | [View Project](https://github.com/TanishaMitra18/Mini-Projects/tree/Heart-Disease-EDA/Heart-Disease-EDA)
- Pharma Commercial Analytics (SQL + Snowflake + Tableau): Built a relational data pipeline in Snowflake joining transaction, provider, and product data, then analyzed HCP value across specialties and channels in Tableau, uncovering that raw revenue rankings flip once normalized per HCP, and that email opt-in HCPs are just as valuable as opt-out ones despite being a tiny fraction of the base. | [View Project](https://github.com/TanishaMitra18/Mini-Projects/tree/Pharma-Commercial-Analytics/Pharma-Commercial-Analytics)
- (future branches will be added here)

## Purpose
The repo serves as a learning playground and portfolio of practical mini-projects, showing how different tools can be applied to solve real-world problems.
The goal is to showcase how everyday problems can be solved with simple code, smart tools, and clear data insights.
